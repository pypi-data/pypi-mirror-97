.. _contribute:

How to contribute
=================

If after using or testing *protopipe* you find that something is missing,
wrong, or simply doesn't do what you expect, you are invited to join the
development effort.

This section contains,

- information on how the project is organized (:ref:`gitrepo`),
- instructions on how to start adding your contribution (:ref:`instructions`),
- what to check before adding it (:ref:`beforepushing`).
